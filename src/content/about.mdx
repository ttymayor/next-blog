# 關於這個部落格

這個部落格是使用最新的 Web 技術建立的。

## 技術堆疊

這個專案使用了以下技術：

- **Next.js 16** - React 框架
- **MDX** - Markdown + JSX
- **Tailwind CSS** - 實用優先的 CSS 框架
- **TypeScript** - 型別安全的 JavaScript
- **next-themes** - 深色模式支援

## 為什麼選擇 MDX？

MDX 結合了 Markdown 的簡潔性和 React 的強大功能。這讓我們可以：

1. 輕鬆撰寫內容
2. 嵌入互動式組件
3. 保持程式碼的可維護性

這個部落格的目標是：

- 分享技術知識
- 記錄學習過程
- 與社群互動
