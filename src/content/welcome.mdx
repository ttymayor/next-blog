# 歡迎來到我的部落格

這是我使用 **Next.js** 和 **MDX** 建立的部落格！

## 什麼是 MDX？

MDX 是一個強大的工具，它允許你在 Markdown 中使用 JSX。這意味著你可以：

- 撰寫 Markdown 內容
- 嵌入 React 組件
- 創建互動式內容

## 功能特色

這個部落格具有以下特色：

1. **深色模式支援** - 自動適應系統主題
2. **響應式設計** - 在所有設備上都能完美顯示
3. **MDX 支援** - 可以在文章中使用 React 組件
4. **快速載入** - 使用 Next.js 的靜態生成
